# 密码管理器

一个简单实用、操作直观的密码管理器，具有现代化的用户界面和完整的功能。

## 功能特性

### 🔐 安全认证
- 用户注册和登录系统
- 主密码加密存储
- JWT令牌认证
- 自动登录状态保持

### 📱 密码管理
- 添加、编辑、删除密码条目
- 按分类组织密码
- 密码加密存储
- 一键查看密码（需要主密码验证）

### 🏷️ 分类管理
- 预设分类：社交媒体、邮箱、银行金融、购物网站、工作相关、其他
- 自定义分类功能
- 分类颜色标识

### ⚙️ 设置功能
- 修改主密码
- 所有密码条目自动重新加密

### 📱 移动端支持
- 响应式设计
- 触摸友好的界面
- 二维码分享功能

## 技术栈

### 后端
- **Node.js** - 运行环境
- **Express** - Web框架
- **bcryptjs** - 密码加密
- **crypto-js** - AES加密
- **jsonwebtoken** - JWT认证
- **qrcode** - 二维码生成

### 前端
- **原生JavaScript** - 无框架依赖
- **CSS3** - 现代化样式
- **Font Awesome** - 图标库
- **响应式设计** - 移动端适配

## 安装和运行

### 1. 安装依赖
```bash
npm install
```

### 2. 配置环境变量
创建 `.env` 文件：
```env
PORT=3000
JWT_SECRET=your-super-secret-jwt-key-change-this-in-production
APP_URL=http://localhost:3000
```

### 3. 启动应用
```bash
# 开发模式
npm run dev

# 生产模式
npm start
```

### 4. 访问应用
打开浏览器访问 `http://localhost:3000`

## 使用说明

### 首次使用
1. 点击"注册"标签
2. 输入用户名和主密码
3. 注册成功后切换到"登录"标签
4. 使用注册的用户名和主密码登录

### 添加密码
1. 点击"添加密码"按钮
2. 填写密码信息（标题、用户名、密码等）
3. 选择分类
4. 输入主密码进行加密
5. 保存

### 查看密码
1. 点击密码卡片上的"眼睛"图标
2. 输入主密码
3. 查看解密后的密码

### 编辑密码
1. 点击密码卡片上的"编辑"图标
2. 修改需要更新的信息
3. 输入主密码
4. 保存更改

### 分类管理
1. 点击侧边栏的"+"按钮添加新分类
2. 选择分类名称和颜色
3. 在添加密码时选择自定义分类

### 修改主密码
1. 点击右上角的"设置"按钮
2. 输入当前主密码
3. 输入新主密码和确认密码
4. 保存更改（所有密码会自动重新加密）

### 分享应用
1. 点击右上角的"分享"按钮
2. 扫描生成的二维码
3. 朋友可以通过二维码直接访问应用

## 安全特性

### 密码加密
- 主密码使用bcrypt进行哈希加密
- 所有密码条目使用AES加密
- 加密密钥为主密码，确保只有用户知道

### 数据传输
- 所有API请求使用HTTPS（生产环境）
- JWT令牌用于身份验证
- 密码在传输前进行加密

### 存储安全
- 密码以加密形式存储
- 主密码哈希存储，无法逆向解密
- 支持主密码修改时重新加密所有数据

## 部署说明

### 本地部署
1. 确保Node.js版本 >= 14
2. 安装依赖：`npm install`
3. 配置环境变量
4. 启动服务：`npm start`

### 云端部署
1. 选择合适的云平台（如Vercel、Heroku、阿里云等）
2. 设置环境变量
3. 部署Node.js应用
4. 配置域名和SSL证书

### 生产环境注意事项
- 修改JWT_SECRET为强密钥
- 配置HTTPS
- 设置合适的CORS策略
- 考虑使用数据库替代内存存储
- 添加日志记录
- 配置错误监控
